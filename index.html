AI — Social Media & Newsletter Writer

Paste an Anthropic API key to turn mentor event details into polished, platform-tailored copy for Instagram, Facebook, X/Twitter, LinkedIn, and the newsletter automatically (model: Claude Opus 4.8). Leave blank to use the built-in templates instead.

ℹ️ How to set up the API key
  1. Go to console.anthropic.com and sign in (or create a free account).
  2. Add a payment method under Settings → Billing (generating posts costs roughly a few cents each).
  3. Open API Keys → Create Key, name it (e.g. “ReGroup App”), and copy the key — it starts with sk-ant-.
  4. Paste it in the box below and click Save AI Key.

Tip: copy the key right away — Anthropic only shows it once. You can delete/rotate it anytime in the console.

The key is stored only in this browser (localStorage) and is sent directly to Anthropic when generating a post. Use a key scoped for this purpose.

🤖 Meeting Bot

Paste the URL of your deployed Meeting Bot backend (see meeting-bot-server/). Once set, calendar events with a video link show a “Send Bot To This Meeting” button — the bot joins, records, summarizes, and messages attendees.

When on, the app dispatches a bot automatically for each calendar meeting with a video link — no manual click. Each dispatch uses Recall credits.

📅 Google Calendar Sync

Paste a Google OAuth Client ID (one-time, free, set by an admin). Then each staff member can connect their own @tjcoregon.org Google Calendar from the Calendar tab — their events show in the app calendar.

ℹ️ How to get the Client ID (≈5 min)
  1. console.cloud.google.com → create/select a project.
  2. APIs & Services → Library → enable Google Calendar API.
  3. OAuth consent screen → Internal (for tjcoregon.org) → add the calendar.readonly scope.
  4. Credentials → Create credentials → OAuth client ID → Web application.
  5. Under Authorized JavaScript origins add https://cameronrthayes-star.github.io.
  6. Copy the Client ID (ends in .apps.googleusercontent.com) and paste it above.
