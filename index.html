AI — Social Media & Newsletter Writer

Paste an Anthropic API key to turn mentor event details into polished, platform-tailored copy for Instagram, Facebook, X/Twitter, LinkedIn, and the newsletter automatically (model: Claude Opus 4.8). Leave blank to use the built-in templates instead.

ℹ️ How to set up the API key
  1. Go to console.anthropic.com and sign in (or create a free account).
  2. Add a payment method under Settings → Billing (generating posts costs roughly a few cents each).
  3. Open API Keys → Create Key, name it (e.g. “ReGroup App”), and copy the key — it starts with sk-ant-.
  4. Paste it in the box below and click Save AI Key.

Tip: copy the key right away — Anthropic only shows it once. You can delete/rotate it anytime in the console.

The key is stored only in this browser (localStorage) and is sent directly to Anthropic when generating a post. Use a key scoped for this purpose.
